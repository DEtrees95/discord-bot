# Commands

## Prefix

Alle Commands starten mit `!`

## Verfügbare Commands

### `!ping`

Antwortet mit `Pong!` – testet ob der Bot online ist.

**Verwendung:** `!ping`

**Antwort:** `Pong!`

---

### `!help`

Zeigt alle verfügbaren Commands an.

**Verwendung:** `!help` oder `!help <command>`

---

## Commands hinzufügen

Neue Commands kommen in `src/cogs/` als eigene Datei.
Danach hier in dieser Datei dokumentieren.
