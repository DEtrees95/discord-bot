\# Installation



\## Voraussetzungen



\- Python 3.11+

\- Git

\- Discord Account + Bot Token



\## Schritt 1 – Repository klonen



```bash

git clone https://github.com/DEtrees95/discord-bot

cd discord-bot

```



\## Schritt 2 – Abhängigkeiten installieren



```bash

pip install -r requirements.txt

```



\## Schritt 3 – Token konfigurieren



`.env` Datei erstellen:



```bash

DISCORD\_TOKEN=dein\_token\_hier

```



!!! warning "Wichtig"

&#x20;   Die `.env` Datei niemals in Git einchecken!



\## Schritt 4 – Bot starten



```bash

python src/bot.py

```
