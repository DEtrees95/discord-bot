\# D\_BOT\_DEV



Ein Discord Bot entwickelt mit Python und discord.py.



\## Features



\- Modular aufgebaut mit Cogs

\- Einfach erweiterbar

\- Vollständig dokumentiert



\## Quick Start



```bash

git clone https://github.com/DEtrees95/discord-bot

cd discord-bot

pip install -r requirements.txt

python src/bot.py

```
